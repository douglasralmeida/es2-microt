Módulo 1 - Usuários
===================
BD: 
 tabela mt_usuarios
  id_usuario - ID do usuário [INT, PK, NOT NULL]
  apelido    - apelido/login [VARCHAR(16), NOT NULL]
  nome       - nome completo [VARCHAR(64), NOT NULL]
  bio        - biografia curta [VARCHAR(128)]
 tabela mt_seguindo
  idseguidor - ID do usuário seguidor [INT, FK, NOT NULL]
  idseguido  - ID do usuário seguido  [INT, FK, NOT NULL]

URLS:
 url/usuario/registrar       - URL para cadastrar usuario
 url/usuario/abandonar/@id   - URL para remover o usuário @id
 url/usuario/@id/seguir	     - URL para seguir usuário @id
 url/usuario/@id/deixar	     - URL para deixar de seguir o usuário @id
 url/usuario/@id/seguidos    - URL para retornar aqueles que o usuário @id segue

Módulo 2 - Mensagens
====================
BD:
 tabela mt_mensagens
  id_msg    - ID da mensagem [INT, PK, NOT NULL]
  id_us	    - ID do usuário que a postou [INT, NOT NULL]
  conteudo  - Conteúdo da mensagem [VARCHAR(140), NOT NULL]

URLS:
 url/msg/postar      - URL para postar uma mensagem
 url/msg/@id         - URL para recuperar a mensagem @id
 url/umsg/@id         - URL para recuperar as mensagens do usuário @id

Módulo 3 - Linha do Tempo
=========================
URLS:
  url/lt/usuario/@id/postagens    - URL para recuperar todas as mensagens do usuário @id
  url/lt/usuario/@id/mural	  - URL para recuperar todas as mensagens do usuário @id e de todos aqueles que ele segue
